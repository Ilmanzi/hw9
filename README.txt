Welcome To EXPRESS JS apps

First thing to do is make sure you have installed all the modules
->
npm install

The database is running on PostgreSQL and try to restore the .sql file
->
movies-database.sql

run the apps with this code inside the terminal :
->
npm run yokbisayok

Open the server :
http://localhost:3000/api-docs/

or you can go to Mover folder to access the endpoint.
Example
->
http://localhost:3000/login

Remember to login, because every endpoints need a token
